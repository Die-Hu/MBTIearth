# Contributing to MBTIearth

Thanks for your interest in contributing! Here's how you can help.

## Reporting Bugs

Found something broken? Open an issue and describe what happened, what you expected, and how to reproduce it.

## Suggesting Features

Have an idea? Open an issue and tell us about it. We'd love to hear your thoughts.

## Submitting Pull Requests

1. Fork the repository
2. Create a new branch for your changes
3. Make your changes and commit them
4. Open a pull request with a clear description of what you did

## Code Style

- Keep it simple -- vanilla JS, HTML, and CSS
- No build tools or frameworks needed
- Write clean, readable code

## Be Respectful

This is a fun project. Be kind to each other and keep things positive.

## A Quick Note

MBTI personality type assignments in this project are just for fun and entertainment. They are not scientifically rigorous or meant to be taken seriously. Enjoy!
